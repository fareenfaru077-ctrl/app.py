Project Overview
This project is a basic REST API developed using Flask (Python framework).
It demonstrates how to build a simple backend application that handles HTTP requests.

The API allows user management operations like:

Create user
View users
Update user
Delete user
Data is stored temporarily using an in-memory dictionary (no database used).

Technologies Used
Python 3
Flask
JSON
Postman (for testing)
⚙️ How to Run the Project
1️⃣ Install Flask (pip install flask) 2️⃣ Run the Application (python app.py)

The server will start at:

http://127.0.0.1:5000
(Welcome to User API)
